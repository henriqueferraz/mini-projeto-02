# mini-projeto

Projeto base em Python.

## Requisitos

- Python 3.10+

## Instalação

```bash
python3 -m venv .venv
source .venv/bin/activate  # Linux/macOS
# .venv\Scripts\activate   # Windows

pip install -e ".[dev]"
```

## Uso

```bash
mini-projeto
# ou
python -m mini_projeto
```

## Testes

```bash
pytest
```
